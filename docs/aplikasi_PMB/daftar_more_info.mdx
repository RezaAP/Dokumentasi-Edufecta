# 2) Proses Daftar Untuk More info

## 1). Masuk ke Halaman More Info
Dengan menggunakan broser pilihan anda, akses link di bawah ini :
```js
http://pmb.edu.educampus.id/
```
<br/>
Kemudian, klik `Info Lebih Lanjut`
