## Setup ID Finger
Fitur Setup ID Finger digunakan untuk mendaftarkan ID yang terdaftar pada mesin fingerprint ke dalam sistem SDM. Proses ini dilakukan agar data presensi setiap pegawai dapat terintegrasi secara otomatis ke dalam sistem, sehingga tidak perlu input manual dan memastikan keakuratan data kehadiran. Untuk menambahkan ID Finger, ikuti langkah berikut:

1). Pada tampilan utama menu Setup ID Finger, klik salah satu nama pegawai. Kemudian isi bagian ID Finger. Anda tidak perlu klik ``Simpan Data`` karena sistem akan otomatis menyimpan data ID Finger yang telah Anda masukkan.

## Proses Presensi

<img
  src="/img/edufecta_Screenshot/SDM/presensi/6.png"

1). Fitur ini untuk melakukan filter data presensi pegawai berdasarkan Harian, Mingguan, Bulanan, dan Tahunan. Anda bisa memilih filter sesuai kebutuhan Anda.
<br style={{ display: 'none' }} />
2). Tombol pada bagian ini digunakan untuk Mencetak Data Presensi, Menghapus Data Presensi, dan Menginput Manual Presensi Pegawai.
<br style={{ display: 'none' }} />
3). Fitur pencarian yang dapat digunakan untuk melakukan pencarian data presensi pegawai berdasarkan Nama.
<br style={{ display: 'none' }} />
4). Fitur ini untuk menentukan tanggal atau bulan atau tahun sesuai dengan filter yang Anda pilih pada nomor 1.
<br style={{ display: 'none' }} />
5). Tabel data yang berisi presensi pegawai yang ada pada sistem. Data akan berubah sesuai dengan filter yang Anda pilih pada nomor 1 dan rentang waktu yang Anda pilih pada nomor 4.

Lalu jika tidak menggunakan mesin fingerprint, bagaimanakah cara menginput manual presensi pegawai ? Berikut langkah-langkahnya:
### Input Manual Presensi Pegawai
a). Klik tombol Tambah Data untuk membuka form input manual presensi pegawai.
<img
  src="/img/edufecta_Screenshot/SDM/presensi/7.png"

  style={{
    border: '1px solid rgb(221, 226, 230)',

    maxWidth: '100%',

b). Isi form input sesuai dengan data pegawai yang ingin Anda inputkan. Pastikan untuk mengisi semua kolom yang diperlukan, seperti Nama Pegawai, Tanggal Presensi, Jam Masuk, Jam Pulang. Setelah selesai, klik tombol Simpan Data.

## Laporan Presensi
Pada menu ini, Anda dapat melihat laporan presensi pegawai berdasarkan filter yang telah ditentukan. Anda bisa melakukan pencarian berdasarkan Nama Pegawai, Tanggal Presensi, dan juga Jam Masuk/Pulang. Laporan ini akan membantu Anda dalam memantau kehadiran pegawai secara efektif. Anda juga dapat mencetak data laporan presensi ini jika diperlukan.
