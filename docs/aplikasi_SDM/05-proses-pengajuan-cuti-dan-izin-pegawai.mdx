# 5.) Proses Pengajuan Cuti dan Izin Pegawai
<div style={{ display: 'flex', gap: '16px', alignItems: 'center' }}>
  <img src="/img/edufecta_Screenshot/SDM/cuti_izin/1.png" alt="Contoh 1" style={{ width: '50%' }} />
  <img src="/img/edufecta_Screenshot/SDM/cuti_izin/2.png" alt="Contoh 2" style={{ width: '50%' }} />
</div>
Fitur ini digunakan untuk mengajukan cuti dan izin pegawai. Proses pengajuan cuti dan izin pegawai penting untuk memastikan bahwa setiap pegawai memiliki catatan yang jelas mengenai absensi mereka. Untuk mengajukan cuti atau izin, ikuti langkah berikut:
:::tip[TIPS]
<strong>Sebelum mengajukan cuti atau izin, pastikan Anda telah melengkapi data master Cuti dan Izin di sistem SDM.</strong>
:::

## Pengisian Master Cuti
Master Cuti digunakan untuk mengatur jenis-jenis cuti yang tersedia di instansi Anda. Proses ini penting untuk memastikan bahwa setiap pegawai memiliki akses ke jenis cuti yang sesuai dengan kebijakan instansi.

  src="/img/edufecta_Screenshot/SDM/cuti_izin/3.png"

    maxWidth: '100%',
    height: 'auto',
    marginBottom: '30px'
  }}
/>
 Untuk mengisi master cuti, ikuti langkah berikut:
1). Klik Tambah Data untuk membuka form input master cuti.
<img
  src="/img/edufecta_Screenshot/SDM/cuti_izin/4.png"
  alt="pilih master cuti"
  style={{
    border: '1px solid rgb(221, 226, 230)',
    borderRadius: '4px',
    maxWidth: '100%',
    height: 'auto',
    marginBottom: '30px'
  }}
/>
2). Input semua form sesuai dengan data instansi Anda, kemudian klik Simpan Data.
<img
  src="/img/edufecta_Screenshot/SDM/cuti_izin/5.gif"
  alt="input master cuti"
  style={{
    border: '1px solid rgb(221, 226, 230)',
    borderRadius: '4px',
    maxWidth: '100%',
    height: 'auto',
    marginBottom: '30px'
  }}
/>
:::tip[TIPS]
Apa itu <strong>Hitung Sisa Cuti</strong> ?
Jika diaktifkan, sistem akan menghitung sisa cuti pegawai secara otomatis berdasarkan jenis cuti yang telah diambil.
:::

## Proses Pengajuan Cuti Pegawai
Sekarang, login ke dalam SDM sebagai pegawai untuk mengajukan cuti. Proses ini penting untuk memastikan bahwa setiap pegawai memiliki catatan yang jelas mengenai absensi mereka. Untuk mengajukan cuti, pada panduang ini, kami akan menggunakan akun pegawai dsengan nama "**ALI YUSUF**" sebagai contoh.
<img
  src="/img/edufecta_Screenshot/SDM/cuti_izin/8.png"
  alt="login pegawai"
  style={{
    border: '1px solid rgb(221, 226, 230)',
    borderRadius: '4px',
    maxWidth: '100%',
    height: 'auto',
    marginBottom: '30px'
  }}
/>
 Jika sudah berhasil login, ikuti langkah berikut:
1). Klik menu **Pengajuan Cuti** dan klik **Tambah Data** untuk membuka form pengajuan cuti.
<img
  src="/img/edufecta_Screenshot/SDM/cuti_izin/9.png"
  alt="tambah data pengajuan cuti"
  style={{
    border: '1px solid rgb(221, 226, 230)',
    borderRadius: '4px',
    maxWidth: '100%',
    height: 'auto',
    marginBottom: '30px'
  }}
/>
2). Pilih jenis cuti yang akan diajukan, kemudian isi form sesuai dengan data cuti yang akan diajukan. Jika sudah, klik tombol **Simpan Data**.
<img
  src="/img/edufecta_Screenshot/SDM/cuti_izin/9.gif"
  alt="pilih jenis cuti"
  style={{
    border: '1px solid rgb(221, 226, 230)',
    borderRadius: '4px',
    maxWidth: '100%',
    height: 'auto',
    marginBottom: '30px'
  }}
/>
3). Sekarang, login sebagai atasan. Pada contoh ini, kami akan menggunakan akun atasan dari **ALI YUSUF** dengan nama "**AGUS RAHMAN ALAMSYAH**". Proses ini penting untuk memastikan bahwa setiap pengajuan cuti pegawai dapat disetujui atau ditolak oleh atasan. Jika sudah berhasil login, klik menu **Pengajuan Cuti**.
<img
  src="/img/edufecta_Screenshot/SDM/cuti_izin/10.png"
  alt="pilih jenis cuti"
  style={{
    border: '1px solid rgb(221, 226, 230)',
    borderRadius: '4px',
    maxWidth: '100%',
    height: 'auto',
    marginBottom: '30px'
  }}
/>
Disini, bisa dilihat bahwa terdapat pengajuan cuti dari **ALI YUSUF** yang sebelumnya telah diajukan. Klik **Detail** / logo mata kuning untuk melihat detail pengajuan cuti.
<img
  src="/img/edufecta_Screenshot/SDM/cuti_izin/11.png"
  alt="pilih jenis cuti"
  style={{
    border: '1px solid rgb(221, 226, 230)',
    borderRadius: '4px',
    maxWidth: '100%',
    height: 'auto',
    marginBottom: '30px'
  }}
/>
Kembali ke halaman sebelumnya, anda bisa klik **Setujui** untuk menyetujui pengajuan cuti pegawai, atau klik **Tolak** untuk menolak pengajuan cuti pegawai. Klik **Ubah Status** untuk mengubah status pengajuan cuti pegawai.
<img
  src="/img/edufecta_Screenshot/SDM/cuti_izin/12.png"
  alt="setujui atau tolak pengajuan cuti"
  style={{
    border: '1px solid rgb(221, 226, 230)',
    borderRadius: '4px',
    maxWidth: '100%',
    height: 'auto',
    marginBottom: '30px'
  }}
/>
Dapat dilihat bahwa status pengajuan cuti pegawai telah berubah menjadi **Disetujui**.
:::tip[TIPS]
 Anda juga bisa mengubah status pengajuan cuti pegawai menjadi **Ditolak** atau sebaliknya.
:::

4). Sekarang, login kembali sebagai pegawai SDM untuk melihat status pengajuan cuti. Jika sudah berhasil login, klik menu **Pengajuan Cuti**. 
<img
  src="/img/edufecta_Screenshot/SDM/cuti_izin/13.png"
  alt="pengajuan cuti pegawai"
  style={{
    border: '1px solid rgb(221, 226, 230)',
    borderRadius: '4px',
    maxWidth: '100%',
    height: 'auto',
    marginBottom: '30px'
  }}
/>

Anda akan melihat bahwa status pengajuan cuti pegawai telah berubah menjadi **Disetujui** oleh atasan dari **ALI YUSUF**. Sekarang klik **Setujui** atau **Tolak** untuk mengubah status pengajuan cuti **ALI YUSUF** oleh **SDM**

5). Kemudian, login kembali sebagai **ALI YUSUF** untuk melihat status pengajuan cuti. Jika sudah berhasil login, klik menu **Pengajuan Cuti**. Dapat dilihat bahwa status pengajuan cuti pegawai telah berubah menjadi **Disetujui** oleh atasan dari **ALI YUSUF** serta dari **SDM**.

  src="/img/edufecta_Screenshot/SDM/cuti_izin/14.png"
