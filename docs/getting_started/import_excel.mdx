a). Menggunakan browser yang anda pilih, akses link di bawah ini :

## 4. Import Data
a). Pada halaman ini, anda dapat menambah data baru dengan menggunakan template excel yang sebleumnya telah ada di sistem.
Pilih salah satu data yang akan anda tambahkan. Selanjutnya, klik **`DOWNLOAD`** yang letaknya pada text "Template Excel".

:::caution[Perhatian]
Pada panduan ini, akan menggunakan `template_programstudi`. Anda dapat menggunakan lainnya, dengan langkah yang sama dengan panduan ini.
:::
